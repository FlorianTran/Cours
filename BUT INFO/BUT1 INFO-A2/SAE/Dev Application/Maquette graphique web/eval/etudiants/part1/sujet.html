Vous disposez des fichiers suivants :

  • debut.html un formulaire qui demande le nom d'un légume et un commentaire check.php
  • check.php qui vérifie si le légume existe est si il est commentable, si c'est le cas le nom et le commenatire doivent être affichés dans welcome.php, sinon affiche le message "vegetable not suitable" dans error.php
  • error.php pour afficher les erreurs et retourner sur la page debut.html
  • welcome.php pour afficher le nom et le commentaire mais ne peut-être accèdé qu'après une authentification réussie en cas d'erreur error.php affiche le message "access denied".
  • VegetableEntity un légume
  • VagetablesEntities une liste de légumes

Remarque ne pas oublier le(s) required car non gérés par un MVC
